# pyfetch

A fast and simple system info fetcher for Linux, written in Python.

## Usage

### Quick Run (one-liner)
```bash
curl -sL https://raw.githubusercontent.com/Abdulhadi446/pyfetch/master/pyfetch.py | python3
```

### Local Run
```bash
chmod +x pyfetch.py
./pyfetch.py
```

## Features
- OS detection
- Host/Model info
- Kernel version
- Uptime
- Shell
- Resolution
- Desktop Environment
- CPU info
- GPU info
- Memory usage
